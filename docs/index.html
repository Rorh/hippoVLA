hippoVLA

Spatial-Temporal Grounded Vision-Language-Action with Hippocampus-Inspired Memory

hippoVLA combines a spatially enhanced VLM backbone with an explicit short-term memory module — inspired by the hippocampus — to give VLA agents the temporal context they need for long-horizon manipulation.

Code TL;DR Highlights Method Results Videos Citation

TL;DR

Current VLA systems suffer from three bottlenecks: missing 3D geometric priors, missing temporal memory, and weak grounding from semantic reasoning to physical action. hippoVLA addresses these with a three-layer design: a hippocampus-inspired spatiotemporal memory, a spatially enhanced RynnBrain-8B backbone, and 3D-grounded training data.

Highlights

  1. Hippocampus-inspired memory. A short-term memory module aggregates history frames at a tunable sampling interval; on LIBERO it lifts Long from 93.0 → 96.0 and Spatial from 93.2 → 94.4.
  2. Spatially enhanced backbone. RynnBrain-8B improves spatial reasoning and object localization, reaching CALVIN ABC→D Avg. Len. 4.38 (rank 3) when paired with hippoVLA memory.
  3. 3D grounded training. Point/depth-aligned supervision bridges semantic reasoning and physical coordinates.
  4. Modular evaluation. Built on StarVLA, hippoVLA enables clean ablations across backbones, memory configurations, and data settings.

Method

hippoVLA is organized as three composable layers that can be ablated independently.

Layer 1 · Hippocampus Memory

Inject explicit short-term memory into the visual encoder so the model retains object trajectories, tracks interaction history, and reasons over long-horizon tasks. The two key knobs are memory length (how many past frames) and sampling interval (how far apart they are sampled). hippoVLA uses length 5 with interval 10 by default.

Layer 2 · Spatially Enhanced VLM

Replace the generic VLM base with RynnBrain-8B, a spatial foundation model that improves geometric understanding and coordinate-sensitive reasoning — directly addressing the 3D-prior bottleneck.

Layer 3 · 3D Grounded Training Data

Integrate depth and point-based supervision into training so the model learns to align language semantics with physically meaningful coordinates rather than 2D pixels.

Engineering Platform

Implemented on StarVLA for clean backbone replacement, modular integration, and controlled ablation across embodied benchmarks (LIBERO 4-in-1, CALVIN ABC→D).

Roadmap

  1. Stage I — Qwen3-VL + hippoVLA memory: establish the memory module and validate on LIBERO 4-in-1.
  2. Stage II — RynnBrain-8B + hippoVLA memory: swap to a spatially stronger backbone; current best on CALVIN ABC→D.
  3. Stage III — 3D grounded training: add depth / point supervision to close the semantic-to-physical gap.
